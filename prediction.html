Let's Predict

This package provides a machine learning model trained based on experimentally measurements to predict the bandgap energy (Eg) for inorganic materials via the command-line. The program uses the python script to predict the band gaps of materials in the Material Science. The model basically uses two datasets to predict the band gaps. The First one being the Elements which has a list of Elements with its properties Like Atomic Number; Mass Number; Group Number; Family Number;Atomic radius. Moreover the second dataset consists of the compound with all the properties like Atomic number;Atomic weight;Group number;Quantam number to train the model to predict the data.
